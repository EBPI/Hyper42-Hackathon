Hyperpoort message portal - Register

On this page you can register yourself using the QR-code provided. This QR-code has to be scanned with your KVK ID app. When successfully authenticated, you will receive a 'Business Card'. With this 'Business Card' you can authenticate yourself and submit your message on the Hyperportal.
